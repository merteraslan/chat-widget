‚öôÔ∏è Configuration Options

The widget supports the following configuration options:

  • webhookUrl (required): Your webhook endpoint
  • title (optional): Widget title
  • initialMessage (optional): First message displayed
  • sessionId (optional): Session identifier
  • csrfToken (optional): CSRF protection token
  • placeholder (optional): Input placeholder text (default: "Type a message‚Ä¶")
  • openByDefault (optional): Whether widget starts open (default: false)
  • className (optional): Additional CSS classes
  • agentName (optional): Display name for AI agent
